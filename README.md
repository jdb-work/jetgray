Project template layout
-----------------------

The project's structure is laid out as follows

    <proj>
      |
      +- src
          |
          +- main
              |
              +- java
              |
                 // application sources
              |
              +- resources
              |
                 // application resources

Execute the following command to compile and package the project

    ./gradlew build
